================================================================================
                        TROUBLESHOOTING
================================================================================

If something doesn't work:

  1. Run: test-system.bat (or ./test-system.sh)
  2. Check if ports 5173, 8000, 5432 are free
  3. Restart Docker Desktop (if using Docker)
  4. Read AUTOMATION_GUIDE.md
  5. Check logs: docker-compose logs -f

Common Issues:

  - Port in use: Stop other services
  - Docker not starting: Restart Docker Desktop
  - Permission denied: Run chmod +x *.sh (Linux/Mac)
  - Services not responding: Wait 30 seconds
