TO RESET POSTGRESQL PASSWORD (if forgotten):

  1. Open: C:\Program Files\PostgreSQL\15\data\pg_hba.conf
  2. Change: md5 to trust (temporarily)
  3. Restart PostgreSQL service
  4. Run: psql -U postgres
  5. Run: ALTER USER postgres PASSWORD 'newpassword';
  6. Change pg_hba.conf back to md5
  7. Restart service
